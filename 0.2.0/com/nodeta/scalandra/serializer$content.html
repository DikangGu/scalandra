scalandra 0.2.0 API

This document is the API specification for scalandra 0.2.0 API

Class Summary
trait Serializer [T] extends AnyRef
Interface for serializers. These are used to serialize and deserialize data going to and coming from cassandra.
Object Summary
object LongSerializer extends Serializer[Long]
This serializer serializes 64-bit integers (Long) to byte arrays in Big-Endian order.
object NonSerializer extends Serializer[scala.Array[Byte]]
This serializer is used when raw data is handled. It does nothing to data.
object StringSerializer extends Serializer[java.lang.String]
This serializer handles UTF-8 encoded strings
object UUIDSerializer extends Serializer[java.util.UUID]
This serializer serializes A UUID to byte arrays in Big-Endian order.
